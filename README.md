# 📚 Sistema Web - Biblioteca Universidade Cesumar

Este repositório contém o projeto final da atividade MAPA, desenvolvido para a disciplina de Desenvolvimento Web. O sistema foi criado para modernizar o controle de acervo da biblioteca, substituindo planilhas manuais por uma aplicação web robusta.

## 🚀 Tecnologias Utilizadas

- **Linguagem:** Java
- **Backend:** Java Servlets, JSF (JavaServer Faces)
- **Frontend:** JSP (JavaServer Pages), HTML, CSS
- **Gerenciador de Dependências:** Apache Maven
- **Servidor de Aplicação:** Apache Tomcat 7
- **Ambiente de Desenvolvimento:** GitHub Codespaces

## 📋 Funcionalidades

- **Cadastro de Livros:** Formulário JSF com validação de campos obrigatórios e formato de ISBN.
- **Listagem de Livros:** Tabela JSP exibindo o acervo de forma organizada.
- **Exclusão de Livros:** Remoção de registros do acervo via Servlet.

## 🛠️ Como rodar o projeto

O projeto foi estruturado com Maven. Para rodar localmente ou em um Codespace, basta executar o seguinte comando no terminal na raiz do projeto:

```bash
mvn clean package tomcat7:run
